HOW TO RUN LOCALLY
 
 add env variables or update configs in `config/`:
 -
 - HERE_API_KEY
 - DB_USERNAME
 - DB_HOST
 - DB_DATABASE_NAME
 - DB_PASSWORD
 - DB_PORT
 
 run scripts:
 -
 - npm install
 - npm run build
 - npm start

 Documentation:
 -
 Go to https://editor.swagger.io/ and insert the content of `swagger.yml`
